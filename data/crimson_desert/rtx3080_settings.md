# RTX 3080 붉은 사막 최적화 설정 가이드

## RTX 3080 사양 참고
- CUDA 코어: 8704
- VRAM: 10GB GDDR6X
- 메모리 대역폭: 760 GB/s
- TDP: 320W
- 아키텍처: Ampere

## 붉은 사막 그래픽 설정 옵션

### 해상도 및 스케일링
- 해상도: 1920x1080 (FHD), 2560x1440 (QHD), 3840x2160 (4K)
- DLSS: 지원 (퀄리티, 밸런스, 퍼포먼스, 울트라 퍼포먼스)
- FSR: 지원 (AMD FidelityFX Super Resolution)
- 프레임 생성: DLSS 3 지원 (RTX 40 시리즈 전용)

### RTX 3080 추천 설정 (QHD 기준, 60fps 목표)
- 해상도: 2560x1440 (QHD)
- DLSS: 퀄리티 모드
- 텍스처 품질: 울트라
- 그림자 품질: 높음
- 반사: 높음 (레이트레이싱 반사 OFF)
- 후처리: 높음
- 식생 밀도: 높음
- 시야 거리: 높음
- 안티앨리어싱: DLSS 사용 시 TAA 불필요
- 모션 블러: 개인 취향 (성능 영향 적음)
- 레이트레이싱: OFF 권장 (RTX 3080에서는 성능 하락 큼)

### RTX 3080 추천 설정 (FHD 기준, 높은 프레임 목표)
- 해상도: 1920x1080 (FHD)
- DLSS: OFF 또는 퀄리티
- 전체 설정: 울트라
- 레이트레이싱: 반사만 ON 가능
- 예상 프레임: 80-100fps

### RTX 3080 추천 설정 (4K 기준)
- 해상도: 3840x2160
- DLSS: 퍼포먼스 모드 필수
- 텍스처: 울트라 (VRAM 10GB 충분)
- 그림자: 중간
- 반사: 중간
- 레이트레이싱: OFF
- 예상 프레임: 45-60fps

## 추가 최적화 팁
- NVIDIA 제어판: 전원 관리 → "최대 성능 선호"
- GPU 스케줄링: 하드웨어 가속 GPU 스케줄링 ON
- 게임바/오버레이: 불필요한 오버레이 OFF
- 드라이버: 최신 Game Ready 드라이버 권장
- VRAM 모니터링: 10GB 초과 시 텍스처 품질 한 단계 낮추기
- GPU 온도: 83°C 이하 유지 권장
