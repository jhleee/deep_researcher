# 붉은 사막 (Crimson Desert)

## 기본 정보
- 정식 명칭: 붉은 사막 (Crimson Desert)
- 개발사: Pearl Abyss (펄어비스)
- 장르: 오픈월드 액션 어드벤처
- 플랫폼: PC, PS5, Xbox Series X|S
- 출시일: 2025년 3월 (얼리 액세스)
- 엔진: 자체 엔진 (검은 사막 온라인 기반 개량)

## 주의
- Pearl Abyss가 개발한 "검은 사막 (Black Desert)" 시리즈의 스핀오프 작품이다.
- 영문 명칭은 "Crimson Desert"이다.

## PC 시스템 요구사항

### 최소 사양
- OS: Windows 10 64-bit
- CPU: Intel Core i5-8400 / AMD Ryzen 5 2600
- RAM: 16GB
- GPU: NVIDIA GTX 1060 6GB / AMD RX 580 8GB
- VRAM: 6GB
- 저장공간: 80GB SSD

### 권장 사양
- OS: Windows 10/11 64-bit
- CPU: Intel Core i7-10700 / AMD Ryzen 7 3700X
- RAM: 32GB
- GPU: NVIDIA RTX 3070 / AMD RX 6800 XT
- VRAM: 8GB
- 저장공간: 80GB NVMe SSD

### 울트라 사양
- CPU: Intel Core i7-12700K / AMD Ryzen 7 5800X3D
- RAM: 32GB
- GPU: NVIDIA RTX 4080 / AMD RX 7900 XTX
- VRAM: 12GB+
